# Task Manager

Task Manager là ứng dụng web tĩnh dùng để quản lý công việc cá nhân trong bài tập bảo trì phần mềm.

Phiên bản hiện tại: v1.0.0

## Chức năng

- Nhập tên và thêm công việc mới.
- Hiển thị danh sách công việc.
- Đánh dấu công việc đã hoàn thành hoặc chưa hoàn thành.
- Xóa một công việc khỏi danh sách.
- Lưu danh sách công việc bằng localStorage của trình duyệt.
- Hiển thị tổng số công việc và số công việc đã hoàn thành.

## Công nghệ sử dụng

- HTML
- CSS
- JavaScript thuần
- localStorage

## Cách chạy ứng dụng

1. Mở thư mục dự án.
2. Mở tập tin `index.html` bằng trình duyệt.
3. Nhập tên công việc và bấm nút `Thêm` để thêm công việc mới.
